# 清算
#### 什么是清算？
算手续费。

#### 什么时候清算？
生成订单的时候，就已经算好了订单金额和手续费。

# 结算
#### 什么是结算？
往商家商户上加钱。

主要分2步，    
1.加钱到账户金额+冻结金额  // T+0结算  
2.加钱到可用金额(可提现)   // T+1结算
